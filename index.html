Command Reference

Command Inputs Description Example
FD
(FORWARD) 		number Moves the turtle forward, in the direction it is facing, by the specified number of turtle steps. FD 100
BK
(BACK) 		number Moves the turtle backward, i.e., exactly opposite to the direction that it's facing, by the specified number of turtle steps. BACK 150
LT
(LEFT) 		number Turns the turtle counterclockwise by the specified angle measured by a number of degrees (1/360 of a circle). LEFT 180
RT
(RIGHT) 		number Turns the turtle clockwise by the specified angle, measured in degrees (1/360 of a circle). RT 90
HOME   Moves the turtle to the center of the graphics area, i.e., coordinates 0,0.  
PU
(PENUP) 		  Lifts the turtle's pen up so that it leaves no trace when it moves.  
PD
(PENDOWN) 		  Puts the turtle's pen down so that it leaves a trace when it moves.  
CS
(CLEAN) 		  Erases everything that the turtle has drawn on the graphics area (ClearScreen). The turtle's state (position, heading, pen color, etc.) is not changed.  
SETCOLOR number Sets the turtle's color - the color of its pen. Color is expressed as a number.
